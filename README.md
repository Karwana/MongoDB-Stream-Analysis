# 🎬 Movie Analysis

Java application for analyzing movie data using Streams API and MongoDB.

## 🚀 Setup

1. Clone the repository
2. Add your MongoDB connection string in `MongoDBAtlasDownload.java`
3. Run the application

## ✨ Features

- 📅 Count movies by year
- ⏱️ Find longest movie
- 🎭 Calculate unique genres/languages
- ⭐ Identify highest-rated movies
- 🎪 Track actor appearances
- 🔍 Custom filtering with predicates

## 🛠️ Tech Stack

Java 17+ • JUnit 5 • MongoDB
